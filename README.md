# GeoGuessr 國家辨識指南

繁體中文版 GeoGuessr 快速查詢工具，涵蓋 153 個國家的辨識線索。

初版，還有很多地方可以補強，先丟上來當快速參考用。

## 功能

- **153 國** 按區域分類，每國包含相機覆蓋、車輛 Meta、道路特徵、車牌、bollard、電線桿、語言文字等辨識資訊
- **實拍照片** — 車輛 Meta、Bollard 防撞柱、車牌（前後）、道路標誌、電線桿（來源：geodummy.com / geomastr.com）
- **11 章通用 Meta 指南** — 相機世代（含圖片）、Google 街景車、文字辨識、道路標線、Bollard 圖鑑、Rift 裂縫、太陽位置、駕駛方向、電話國碼、TLD、推薦學習資源
- **繁體中文** 介面與內容

## 使用方式

靜態網頁，不需要伺服器。

直接開 `index.html` 或到 **https://jerrylin1213.github.io/geoguessr-guide/**

## 已知不足

- 部分國家資料不夠詳盡（特別是非洲、大洋洲小國）
- 拍攝年份沒有全部標註
- 圖片還可以補更多（特別是電線桿、地景）
- 部分翻譯可能有遺漏
